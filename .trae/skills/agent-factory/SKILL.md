# Agent Factory Skill

Habilidad para generar nuevos agentes especializados de forma autónoma.

## 🚀 Flujo de Ejecución
1.  Detectar necesidad de especialización.
2.  Definir nombre y rol del nuevo agente.
3.  Utilizar `.trae/agents/template-agent.md` para crear el archivo.
4.  Registrar en `AGENTS.md`.
